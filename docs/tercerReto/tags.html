LISTA DE ELEMENTOS HTML5

Algunos ejemplos:

  • Metadatos del documento
  • Secciones
  • Agrupación del contenido

Metadatos del documento

head

Representa una colección de metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo.

Secciones

body

Representa el contenido principal de un documento HTML. Solo hay un elemento body en un documento.

header

Define la cabecera de una página o sección.

main

Define el contenido principal de un documento.

footer

Define el pie de una página o sección.

nav

Define una sección que solamente contiene enlaces de navegación

section

Define una sección en un documento.

h1,h2,h3,h4,h5,h6

Un elemento de cabecera describe brevemente el tema de la sección que introduce.

Agrupación de contenido

p

Define una parte que debe mostrarse como un párrafo.

ul

Define una lista de artículos sin orden.

li

Define un artículo de una lista enumerada.
